Turtle Back Arch and 718 named arches

University of Kentucky Geography

Arch locations are extracted from the Domestic and Antarctic Names (State and Topical Gazetteer) published by the United States Board on Geographic Names. (geonames.usgs.gov). Data accessed August, 2018.

Elevation contours shown have an interval of 20 feet with labeled index contours every 100 feet. Vector data was extracted from a USGS NED 1/3 arc-second raster digital elevation model (approximately 10-meter resolution). This data can be accessed at the USGS 3D Elevation Program (usgs.gov).

Tools used in this project

  • PostGIS was used to store/query the national dataset.
  • Data is shown on Mapbox web map with geolocation services enabled. If the map got popular it would require a fee to use.
  • GDAL and QGIS to convert raster elevation units and create contours.

Project assets

  • Point locations of US arches as GeoJSON file
